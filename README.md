## Featured Projects

### Image Captioning System
An encoder-decoder deep learning model that generates natural language descriptions for images. The system combines a CNN-based encoder (ResNet50) for visual feature extraction with an LSTM-based decoder for sequence generation. Trained on the MS-COCO dataset and optimized for inference latency under 300ms.

### Customer Churn Prediction
A production-ready machine learning system that predicts customer attrition with 87% accuracy using demographic, account, and behavioral data. Implements feature engineering pipelines, class imbalance handling via SMOTE, and hyperparameter optimization. The model is containerized with Docker and exposes a REST API via FastAPI.
